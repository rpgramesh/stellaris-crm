# Style Guide - Stellaris Consulting Theme

## Brand Identity
This design system is inspired by the Stellaris Consulting brand, featuring a professional color palette centered around deep blues, bright accents, and clean typography.

## Color Palette

### Primary Colors
- **Primary Blue**: `#1357bf`
  - Used for: Primary buttons, active states, key headers.
  - Accessibility: Pass AAA (8.5:1) on white background.
- **Primary Foreground**: `#ffffff`
  - Used for: Text on primary buttons.

### Secondary & Accents
- **Secondary**: `#e9ecef` (Light Grey)
  - Used for: Secondary buttons, subtle backgrounds.
- **Accent**: `#037ef3` (Bright Blue)
  - Used for: Highlights, links, icons.
- **Success**: `#4ED199` (Mint Green)
  - Used for: Success states, positive indicators.

### Functional Colors
- **Background**: `#f8f9fa` (Light) / `#0f172a` (Dark)
- **Foreground (Text)**: `#212529` (Dark) / `#f8f9fa` (Light)
- **Destructive**: `#ff6c5f`
- **Warning**: `#ffc107`
- **Info**: `#0dcaf0`

## Dark Mode
The dark theme uses a deep slate palette (`#0f172a`) to reduce eye strain while maintaining brand identity through lighter blue accents (`#4d84e2`).

## Typography
- **Font Family**: Geist Sans (System UI)
- **Headings**: Bold, high contrast.
- **Body**: Regular, high readability.

## Usage Guidelines
- Always ensure sufficient contrast for text.
- Use Primary Blue for the main call-to-action (CTA).
- Use Secondary Grey for cancel buttons or secondary actions.
- Use Accent Blue sparingly to draw attention.
